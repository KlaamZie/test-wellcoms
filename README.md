# Technical test

## Installation

```bash
git clone https://github.com/KlaamZie/test-wellcoms.git
cd test-wellcoms
npm install
```

## Usage

### Compile ts to js

```bash
npm run compile
```

### Run tests

```bash
npm run test
```

### Run the scripts

Run filterDuplicates function

```bash
npm run filter
```

Run getTopStocks function

```bash
npm run stocks
```

Run findSumPair function

```bash
npm run pair
```
